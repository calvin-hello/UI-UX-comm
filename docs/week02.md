## Topics
Three main topics were discussed in class,
1. Design Thinking
2. User-Centered Design
3. Usability Factors

### Definition of Topics

* **Design Thinking** is a problem solving process that can be applied to problems.

* **User-Centered Design** is an iterative set of steps to solve problems related to a product or service 
for users.

### Design Thinking process
1. Empthize
2. Define
3. Ideate - brainstorm
4. Prototype
5. Test

### User-Centered Design process
1. Understand the context
2. Specify requirements
3. Design solutions
4. Evaluate solutions
