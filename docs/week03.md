The topics and in-class activities focused on using **personas**, **user scenarios**, and **user stories** to better understand user needs and guide design decisions.

## Definition of terms

* _**Personas**_: identify user characteristics, behaviours, and goals
* _**User Scenarios**_: a technical description of the steps required to complete one of your user stories.

* _**User Stories**_: identify features (technical features of a product/service that need to be built). It make up a list of tasks that you can use to ensure, at the end of the project, that you have built everything you set out to build
